# 🏡 Vrudhashram Management System

A simple and user-friendly web application to support old age homes by managing donations, volunteers, and requirements.

---

## 📌 Project Overview

This project helps old age homes (Vrudhashram) solve real-world problems like:

- Managing daily needs (food, medicines, clothes)
- Accepting donations from users
- Registering volunteers
- Admin tracking all activities

---

## 🎯 Features

### 👤 User Side
- 🔐 Login / Register system
- 🏠 Home & About pages
- 💝 View needs and donate items
- 🙋 Volunteer registration
- 💰 QR code payment support

### 👨‍💼 Admin Side
- 📊 Dashboard with stats
- ➕ Add / ❌ Delete needs
- 📋 View donations
- 🙋 View volunteers
- 🔐 Role-based access (Admin only)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Database:** PostgreSQL
